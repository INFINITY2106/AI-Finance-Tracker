# AI Finance Tracker

A full-stack personal finance management application built using HTML, CSS, JavaScript, Spring Boot, and MongoDB.

## Features

- Income Tracking
- Expense Management
- Expense Analytics
- Financial Health Score
- Goal Tracking
- AI Insights
- PDF Reports
- Dark Mode
- MongoDB Integration

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript
- Chart.js

Backend:
- Spring Boot
- Java

Database:
- MongoDB

- JWT Authentication
